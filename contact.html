Questions Fréquentes

Comment installer CareLink ?

Téléchargez l'installateur pour votre système d'exploitation, exécutez-le et suivez les instructions. L'installation prend moins de 5 minutes.

Configuration minimale requise ?

Windows: Windows 10+, 4GB RAM, 500MB disque
macOS: macOS 10.15+, 4GB RAM, 500MB disque
Linux: Ubuntu 20.04+ ou équivalent, 4GB RAM, 500MB disque

Mes données sont-elles sécurisées ?

Absolument ! Toutes vos données sont stockées localement sur votre ordinateur avec un chiffrement AES-256. Aucune donnée n'est envoyée sur internet.

CareLink est-il vraiment gratuit ?

Oui, 100% gratuit sans limitation. Aucun abonnement, aucune publicité, aucun coût caché. Toutes les fonctionnalités sont incluses gratuitement.

Y a-t-il une version mobile ?

Pas encore. CareLink est actuellement disponible uniquement sur desktop (Windows, Mac, Linux). Une version mobile est prévue pour l'avenir.

Comment sauvegarder mes données ?

CareLink effectue des backups automatiques quotidiens. Vous pouvez aussi exporter manuellement toutes vos données via la section "Backup" de l'application.

Faut-il une connexion internet ?

Non, CareLink fonctionne entièrement hors ligne. Une connexion internet est nécessaire uniquement pour les fonctionnalités d'IA (mais vous pouvez utiliser Ollama en local).

Combien de membres puis-je ajouter ?

Il n'y a aucune limite ! Vous pouvez ajouter autant de membres de famille que vous le souhaitez, chacun avec son propre profil et ses propres données.

Comment obtenir de l'aide ?

Envoyez-nous un email à Riisalthkarral@gmail.com. Nous répondons généralement sous 24-48h. Vous pouvez aussi consulter la documentation intégrée à l'application.

Prêt à Commencer ?

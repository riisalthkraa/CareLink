Questions Fréquentes

Pourquoi une version payante ?

La version Pro finance le développement continu, les serveurs pour le CI/CD, et permet de proposer un support prioritaire. La version gratuite reste disponible pour tous.

Quels moyens de paiement ?

Carte bancaire, PayPal, virement bancaire. Paiement sécurisé. Facture fournie automatiquement.

Et les mises à jour ?

Version Open Source : mises à jour manuelles. Version Pro : mises à jour automatiques avec toutes les nouvelles fonctionnalités incluses à vie.

Plusieurs utilisateurs ?

Une licence Pro = installations illimitées pour un usage familial ou professionnel personnel. Pour entreprises, contactez-nous pour licence multi-utilisateurs.

Mes données sont-elles sûres ?

Oui, vos données restent sur votre ordinateur (local). Aucun cloud, aucune transmission. Chiffrement AES-256 dans les deux versions.

Remboursement possible ?

Oui, remboursement intégral sous 30 jours sans justification. Vous testez pendant 60 jours, si ça ne convient pas, remboursement garanti.
